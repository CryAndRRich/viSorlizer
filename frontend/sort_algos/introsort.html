Intro sort, or introspective sort, is a hybrid sorting algorithm

It combines the good parts of the three algorithms, begins with quicksort, it switches to heapsort when the recursion depth exceeds a level based on the number of elements being sorted and it switches to insertion sort when the number of elements is below some threshold
