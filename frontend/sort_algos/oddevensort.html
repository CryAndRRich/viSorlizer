Odd-even sort, also known as brick sort or parity sort, is a relatively simple sorting algorithm. It functions by comparing all odd/even indexed pairs of adjacent elements in the list and, if a pair is in the wrong order the elements are switched

The next step repeats this for even/odd indexed pairs. Then it alternates between odd/even and even/odd steps until the list is sorted
