Flash sort is an efficient in-place, distribution sorting algorithm. It' s rearrangement phase operates in cycles. Elements start out "unclassified", then are moved to the correct bucket and considered "classified"

The basic procedure is to choose an unclassified element, find its correct bucket, exchange it with an unclassified element there, mark it as classified, and then repeat with the just-exchanged unclassified element. Eventually, the element is exchanged with itself and the cycle ends
