Stooge sort is a recursive sorting algorithm. The algorithm is defined as follows:

If the value at the start is larger than the value at the end, swap them. If there are three or more elements in the list, then Stooge sort the initial 2/3 of the list, Stooge sort the final 2/3 of the list and Stooge sort the initial 2/3 of the list again
